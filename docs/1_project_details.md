# Project Details: Tailor Shop Management System

## Overview
A web application for managing a tailor shop, supporting complete order lifecycles and user roles (Admin, Tailor, Customer).

## Technology Stack
- **Frontend**: Next.js 14+ (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Backend/Database**: Supabase
- **Internationalization**: i18n (English, Urdu, Arabic)

## Key Features
- **Multi-language Support**: Experience in English, Urdu, and Arabic.
- **User Roles**: Distinct flows for Admin, Tailor, and Customer.
- **Order Management**: Life cycle tracking from Pending to Delivered.
